**Smart Deals** is a full-featured marketplace web application where users can buy products, bid on items, and post their own products for sale.  

Built with **React**, **Node.js**, **Express.js**, **MongoDB**, **TailwindCSS**, **DaisyUI**, and **Axios**, this app provides a seamless and interactive experience for both buyers and sellers.

---

## ğŸŒŸ Features

### **Buyer / User Features**
- Browse and purchase any product.  
- Place bids on products.  
- View all bids made by the user.  
- Delete or update their own bids and products.  

### **Seller Features**
- Post new products for sale.  
- Track who has placed bids on their products.  
- Update or delete their own products.  

### **Real-Time Features**
- Real-time bid updates via backend API and MongoDB.  
- Smooth and responsive UI powered by **TailwindCSS** and **DaisyUI**.
